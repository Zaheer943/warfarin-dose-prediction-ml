## warfarin-dose-prediction-ml
End-to-end machine learning pipeline for personalized Warfarin dose prediction using clinical and genetic data.

## Contents

- Jupyter Notebook (.ipynb) – Full ML workflow
- Python Script (.py) – Pipeline implementation
- PDF Report – Methodology and results

## Project Highlights

- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- ML models: Linear Regression, Lasso, SVR, Random Forest, XGBoost
- Deep Learning: Artificial Neural Network (TensorFlow)
- Model evaluation (MAE, RMSE, R²)
- Feature importance analysis
- Gradio web application for real-time dose prediction

## Dataset

IWPC dataset was obtained from PharmGKB.
